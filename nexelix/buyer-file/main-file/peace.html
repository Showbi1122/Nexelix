The PEACE Group of Schools & Colleges — Website Redesign

Turning a content-heavy, outdated site into a clean, engaging, mobile-first experience that reflects the institution’s prestige and improves admissions discovery.

Role: UI/UX & Dev Stack: Bootstrap 5, SCSS, JS Year: 2025
View Live Site

Background

The PEACE Group is an established education network in KPK. Their previous site was serviceable but visually dated, dense, and hard to navigate—especially on mobile.

  • Inconsistent typography & spacing
  • Cluttered navigation; admissions info hard to find
  • Poor visual hierarchy; low engagement

Goals

  • Modern, trustworthy visual identity aligned with the brand
  • Mobile-first layout with clearer IA
  • Prominent admissions CTAs and campus highlights
  • Embed media (video) to tell the story
  • Fast, accessible, and easy to maintain

My Role

Research, wireframes, high-fidelity UI, front-end development, performance & accessibility pass.

Approach

  1. Audit & IA: mapped key journeys (Admissions, Academics, Campuses) and simplified the menu.
  2. UI System: type scale, spacing, and reusable components (hero, stats, cards, CTA blocks).
  3. Media & Story: embedded YouTube to showcase culture and campus life.
  4. Performance: compressed assets, deferred non-critical JS, semantic HTML.
  5. Accessibility: contrast-safe palette, focus states, alt text, ARIA labels.

Highlights

Simplified Navigation

Clear paths to Admissions, Academics, and Campuses reduce friction.

Video Integration

YouTube embed improves time-on-page and trust.

Mobile-First Layout

Optimized typography, spacing, and tap targets.

Faster Loads

Lazy images + minimized CSS/JS for better Core Web Vitals.

Before → After

Outcome

The redesigned website now communicates the institution’s values clearly, improves discoverability of admissions content, and provides a polished, trustworthy digital identity that supports the group’s growth.
